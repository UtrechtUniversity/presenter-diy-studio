# Presenter DIY Studio
The Presenter DIY Studio is a recording studio in which visitors can independently create professional videos to support teaching and science communication. Visitors do everything themselves, including starting up and shutting down the studio, and are guided throughout the entire process by an explanatory video, instruction posters and an application on the computer that takes the user through all the steps. After recording, the video can be reviewed and, once approved, the integration with OneDrive ensures that videos are uploaded directly to the cloud. Recordings can be made with a photo or solid colour as the background, or with a PowerPoint presentation in the background. If desired, the presenter’s script can be displayed on a screen in front of the camera, so that it can be read while presenting.

The project is open source: from the manual, technical drawings and required assets to the application and required plugins. In terms of required third party software, some these applications are not open source: Microsoft Windows, SharePoint/OneDrive, PowerPoint, Blackmagic Desktop Video, Blackmagic Ultimatte Software Control and the Stream Deck application.

This repository contains the documents and assets for the Presenter DIY Studio in the University Library at Utrecht Science Park.

The studio works in combination with the following application:
* [DIY Studio App](https://github.com/UtrechtUniversity/diy-studio-app)
