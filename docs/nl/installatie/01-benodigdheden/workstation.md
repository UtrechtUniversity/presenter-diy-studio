Een workstation is nodig met grofweg de volgende specificaties:

- Een moderne CPU: Intel Core i5-14400 4.70G 20 MB 10 cores 65W CPU

- 32GB (2x16GB) DDR5 4800 UDIMM Memory

- Een grafische kaart van Nvidia die hardwarematig h.264 kan coderen en
  waar 3 Full HD monitoren op kunnen worden aangesloten (DP of HDMI) -\>
  NVIDIA GeForce RTX 4060 8 GB GDDR6 PCIe x16 Graphics

- 2X 2TB PCIe-4x4 2280 NVMe TLC Solid State Drive

- Twee ethernet-poorten: één voor soliscom/internet, één om te
  communiceren met de Blackmagic Ultimatte hardware keyer (deze heeft
  een statisch IP-adres nodig) -\> Intel I226-T1 2.5GbE Ethernet Network
  Adapter

- Uitbreidingskaart (PCIe): Blackmagic Decklink Studio 4K

- PCI Express (minimaal 2.0) 4x slot voor een Blackmagic Decklink Studio
  4K (deze neemt twee plekken in beslag)

- Fysieke pinnen op het moederbord om de externe aan/uit-knop op aan te
  sluiten
