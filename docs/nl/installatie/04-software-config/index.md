Elk kwartaal zorgt Content&Digital voor een **DIY
Studio Software Package**. In dit ZIP-bestand zullen alle juiste versies van
alle applicaties en drivers zitten alsmede alle benodigde assets en
configuratiebestanden. Deze packages zijn ook te vinden op de netwerkstorage van
Content&Digital in de map *05_DIY-Studio_Packages*.

Uitleg over het updaten van Packages staat in
de Supporthandleiding (linken).

- Kopieer de map *Software* in *Kopieren
  naar C* naar de C-schijf

- Kopieer de map *studio-mediafiles* in *Kopieren naar D* naar de D-schijf.
