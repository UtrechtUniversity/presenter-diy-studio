The metal platform, light and camera tripod are screwed directly to the desk.

Power strip #3 is available for users to charge phones and laptops. Secure it with strong double-sided tape. In the current studio it is attached to a separate table.

Attach a magnetic sticker to the underside of the Stream Deck so that it stays on the metal platform but can still be removed.
