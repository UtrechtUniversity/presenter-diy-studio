A workstation with approximately the following specifications is required:

- A modern CPU: Intel Core i5-14400, 4.70 GHz, 20 MB, 10 cores, 65 W
- 32 GB (2×16 GB) DDR5 4800 UDIMM memory
- An Nvidia graphics card capable of hardware H.264 encoding and supporting three Full HD monitors over DisplayPort or HDMI: NVIDIA GeForce RTX 4060, 8 GB GDDR6, PCIe x16
- 2× 2 TB PCIe 4.0 x4 2280 NVMe TLC solid-state drives
- Two Ethernet ports: one for soliscom/internet and one for communication with the Blackmagic Ultimatte hardware keyer, which requires a static IP address; for example, an Intel I226-T1 2.5GbE network adapter
- Expansion card: Blackmagic DeckLink Studio 4K
- A PCI Express x4 slot, version 2.0 or later, for the Blackmagic DeckLink Studio 4K; the card occupies two expansion slots
- Physical motherboard pins for connecting the external power button
