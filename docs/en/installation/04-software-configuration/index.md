Every quarter, Content & Digital prepares a **DIY Studio Software Package**. This ZIP archive contains the correct versions of all applications and drivers, together with all required assets and configuration files. The packages are also stored on the Content & Digital network storage in the *05_DIY-Studio_Packages* folder.

Instructions for updating packages are included in the support manual; a link still needs to be added.

- Copy the *Software* folder from *Kopieren naar C* to the C drive.
- Copy the *studio-mediafiles* folder from *Kopieren naar D* to the D drive.
