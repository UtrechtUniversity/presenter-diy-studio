Sign in as **User** to change these settings.

The Stream Deck software is configured to show the correct buttons for both the DIY Studio App and PowerPoint.

A custom Stream Deck plugin is included in the DIY Studio Software Package.

- Copy the `nl.uu.diy-studio.sdPlugin` folder to:

  ```text
  C:\Users\User\AppData\Roaming\Elgato\StreamDeck\Plugins
  ```

- Restart the Stream Deck software. A `DIY Studio` category will now appear at the bottom of the right-hand panel, containing:
  - Record-stop toggle
  - Back to first slide
  - Previous slide
  - Next slide
  - Language toggle — **do not use this action**

Drag the actions from the right-hand panel onto the buttons as shown:

![Stream Deck showing, from the top left: button 1, Record-stop toggle; button 3, Back to first slide; button 4, Previous slide; button 5, Next slide](../../media/stream-deck-panel.png)

- Open *Settings* using the cog icon and clear:
  - `Automatically check for updates`
  - `Open automatically at login`
