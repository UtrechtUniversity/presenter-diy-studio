This software package contains the drivers and configuration application required for the DeckLink to work correctly. Configure the application by clicking the circular icon:

![Screenshot: circular document icon below DeckLink Studio 4K](../../media/desktop-video-1.png)

Then use:

- `Default video standard: 1080p25`
- `Video playback: Displays black output when not playing`
