# Presenter DIY Studio

## Record professional videos independently

The Presenter DIY Studio is a recording studio where visitors can independently create professional videos for teaching and science communication. Visitors operate everything themselves, including starting up and shutting down. Throughout the process, they are guided by a tutorial video, instruction posters and a computer application that takes them through every step. Recordings can be reviewed afterwards and, once approved, are uploaded directly to the cloud through the OneDrive integration. You can record with a photograph, a solid colour or a PowerPoint presentation as the background. If required, the presentation text can be displayed on a screen in front of the camera so that it can be read as a teleprompter.

![Photograph of the studio](media/diy-studio-1.jpg)

The project is open source, including the manuals, technical drawings, required assets, application and plugins. Some software components are not open source: Microsoft Windows, SharePoint/OneDrive, PowerPoint, Blackmagic Desktop Video, Blackmagic Ultimatte Software Control and the Stream Deck application.

The project consists of the following GitHub repositories:

- [Main repository containing manuals and assets](https://github.com/UtrechtUniversity/presenter-diy-studio)
- [DIY Studio App](https://github.com/UtrechtUniversity/diy-studio-app)
