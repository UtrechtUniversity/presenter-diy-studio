# 2.4 Other hardware problems

## The microphone does not work

- Speak and check whether the red level LEDs on the vocal processor respond.
- If not, there may be a problem with the XLR cable between the microphone and vocal processor, or with the microphone itself.
- Check whether the camera receives audio, for example by connecting headphones.
- Sign in as administrator, start OBS and check whether its audio meters respond.
- Check the cables and replace them if necessary.

## The camera does not work

- If the white LED on the front is off, check the power connection. Try switching the camera on manually by holding the power button to the left of the lens for one second.
- If the white LED is on, check the SDI cables from the camera to the Ultimatte.

The camera must be connected using:

1. a `micro-BNC (male) -> BNC (female)` cable
2. followed by a `BNC (male) -> BNC (male)` cable connected to the Ultimatte's `CAMERA FG` input

Disconnect and reconnect the cables, or replace them.

## The control panel does not work

The panel should look like this:

![Stream Deck showing, from the top left: button 1, Record-stop toggle; button 3, Back to first slide; button 4, Previous slide; button 5, Next slide](../../media/stream-deck-panel.png)

- If the display is blank, check the USB connection and whether the Stream Deck software is running.
- If other icons are shown, configure the correct buttons in the Stream Deck software.
- Close the DIY Studio App with `Alt+F4`.
- Open the Stream Deck software, which should be visible in a minimised window on the left-hand display.

![Stream Deck application with the required button layout](../../media/stream-deck-app.png)

- Drag the correct actions to the correct positions.
- If the DIY Studio plugin is absent from the right-hand panel, install it as described in the installation manual.
- If the panel looks correct but the buttons still do not work, check Command Prompt for errors. Photograph the error, send it to Content & Digital and restart the PC.

## The computer does not shut down

Videos may still be uploading or Windows updates may be installing. The computer will shut down automatically afterwards. If not, it will shut down after 60 minutes.

If the PC still does not shut down, hold the power button for eight seconds to switch it off manually.
